A modern, feature-rich admin dashboard for managing business operations, including products, customers, sales transactions, analytics, and user management. Built with vanilla JavaScript, Chart.js, and GSAP for smooth animations.

## ✨ Features

### Core Modules
- **Dashboard** - Real-time KPI cards with count-up animations, interactive charts (revenue trends, category distribution, sales performance, top products), and recent transactions list.
- **Product Management** - Complete CRUD operations, category filtering, search, sorting, pagination, and stock level indicators.
- **Customer Management** - Manage customer database with search and pagination.
- **Transaction Recording** - Log new sales transactions, view history with date filtering and search.
- **Reporting System** - Generate dynamic reports (Revenue, Product, Customer, Inventory) with CSV export and print capabilities.
- **Data Import** - Drag-and-drop CSV/TXT import with file preview and import history.
- **User Management** - Manage system users with role-based access (Admin, Manager, Staff).
- **Settings** - Customizable company info, currency format, dark mode, and notification preferences.

### Technical Highlights
- **Dual Theme** - Light/Dark mode with smooth transitions and persistent settings.
- **Responsive Design** - Mobile-friendly sidebar, tables, and charts.
- **Client-Side Database** - Built-in mock database (easily replaceable with a backend API).
- **Interactive Charts** - Dynamic charts using Chart.js (line, bar, doughnut).
- **Smooth Animations** - GSAP-powered entrance animations and micro-interactions.
- **Toast Notifications** - User feedback system for all actions.
- **Pagination** - Built-in pagination for all data tables.
- **Search & Filter** - Real-time search and category filtering across modules.
